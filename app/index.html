Overlay Editor Beta Demo
This workflow previews World Explorer overlay editing on top of the imported world. It is not connected to OpenStreetMap contribution and it does not edit base ingest data.
1. Select a base feature to patch, or choose a preset and draw a point, line, or polygon directly in the 3D world.
2. Edit vertices, tags, and 3D properties like height, levels, roof shape, bridges, tunnels, and entrances.
3. Validate the overlay, preview it locally, and save draft revisions without affecting the public world.
4. Submit the overlay for moderation. Admin review decides whether it needs changes, is rejected, or is approved and published.
5. Published overlays merge into runtime render and traversal layers without mutating the imported base dataset.

🚗 Driving Mode:

🚤 Boat Mode:
• W/S - Throttle / Reverse
• A/D - Steer
• Space - Brake / Hold position
• G - Enter / exit boat travel
• C - Camera view
• Use the bottom Wave Intensity slider for calm, moderate, or rough water
🚶 Walking Mode:
• W/S - Move forward / reverse
• A/D or Arrow Left/Right - Turn camera
• Arrow Up/Down - Look up / down
• E - Enter/exit building interior
• Space - Jump
• Shift - Run (2x speed)
• Right Click + Drag - Mouse look

• F - Toggle drive mode
• 6 - Toggle drone mode
• B - Toggle build mode
• Click place block, Shift+Click remove

• WASD - Move
• Arrow Keys - Look around
• Mouse - Look around
• Space - Fly up
• Shift/Ctrl - Fly down

• Click place block, Shift+Click remove
